- [x] Ask questions and get answers
- [x] Stay in touch with DSC-VIT

# Contributing

## Pre-requisites:

- Flutter
- Dart SDK >=2.7.0 <3.0.0

## Getting started

```bash
git clone https://github.com/GDGVIT/devstack-app.git
```

## Build and run

```bash
flutter run
```
